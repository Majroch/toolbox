# ToolBox
*Author: Majroch <jakuboch4@gmaill.com>*

### Usage
Download repository to your PC:
```bash
git clone https://github.com/Majroch/toolbox.git
```

Run "first time" script:
```bash
cd toolbox/
./init_server.sh
```
or
```bash
cd toolbox/
composer i
bower i
```

Start server:
```bash
./start_server.sh
```
or
```bash
bin/console server:start *:8000
```
